# Sugestões de evolução para sistema web

## Caminho mínimo

- Google Forms para coleta.
- Google Sheets como base de consolidação.
- Script de importação para gerar os JSONs do site.
- Publicação automática em GitHub Pages.

## Caminho intermediário

- Formulário web próprio.
- Upload da planilha preenchida.
- Validação automática no backend.
- Consolidação por apostador e por fase.
- Atualização automática do ranking.

## Caminho completo

- Painel administrativo.
- Cadastro de apostadores.
- Pagamento com conciliação.
- Resultados oficiais sincronizados por API.
- Chaveamento e classificação em tempo real.
- Histórico por fase e por edição.

## Recomendação prática

- Manter o Excel como interface de entrada por enquanto.
- Centralizar a consolidação em uma fonte única.
- Evoluir para web assim que o volume de participantes justificar a migração.
