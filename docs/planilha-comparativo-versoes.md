# Comparativo de versões

## Versão anterior

- Planilha focada apenas na captura básica de palpites da fase mata-mata.
- Identificação simplificada.
- Pouca parametrização.
- Sem estrutura dedicada para pagamento, fonte oficial e consolidação.
- Chaveamento com menor clareza operacional.

## Versão nova

- Identificação reorganizada.
- ID único do apostador.
- Data/hora de envio.
- Controle de pagamento por PIX.
- Status de pagamento e confirmação.
- Fonte oficial e versão explícitas.
- Chaveamento oficial separado.
- Regulamento consolidado.
- Configurações ocultas para parametrização futura.
- Validação de placares.
- Validação de status.
- Preparação para atualização automática.

## Ganhos práticos

- Menos erro de preenchimento.
- Mais rastreabilidade de cada apostador.
- Melhor base para centenas de participantes.
- Melhor compatibilidade com automação futura.
- Mais clareza para quem envia e para quem consolida.

## Observação

- A regra de pontuação continua baseada apenas no placar de 90 minutos.
- Prorrogação e pênaltis servem somente para definir classificado no mata-mata.
